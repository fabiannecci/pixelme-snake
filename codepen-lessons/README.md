# PixelMe CodePen Collection Structure

This directory contains the code for 7 progressive lessons. Each lesson folder contains:
- `index.html` - HTML section for CodePen
- `styles.css` - CSS section for CodePen
- `script.js` - JavaScript section (STARTER code for students)

## How to Use with CodePen

1. Create a new CodePen collection called "PixelMe Course"
2. For each lesson (lesson-1 through lesson-7):
   - Create a new pen in your collection
   - Copy the contents of `index.html` into the HTML panel
   - Copy the contents of `styles.css` into the CSS panel
   - Copy the contents of `script.js` into the JS panel
   - Save with the lesson title

## Lesson Sequence

- **Lesson 1**: Bits and Pixel Rendering
- **Lesson 2**: Variables and Animation  
- **Lesson 3**: Keyboard Controls
- **Lesson 4**: Arrays and Matrices
- **Lesson 5**: Snake Movement and Collision
- **Lesson 6**: Snake Growth
- **Lesson 7**: Game Over and Self-Collision
